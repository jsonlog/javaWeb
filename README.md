Args
==================

Args 是用来获取Web中Action(或Controller)参数的,可以通过反射传入参数调用Action方法,Args的作用是获取参数,不提供其他功能

分支:Smart_args
=======
该分支提供了一个便于smart框架使用的方法类：com.isea.tools.smart.SmartArgsUtil
